PORTFÓLIO

  • PalmTax

    Projeto desenvolvido em python para consulta de tributação de alguns produtos a partir de um banco de dados. É possivel realizar inclusão, exclusão e consulta desses materiais, bem como existe a implementação de redirecionamento para o Google Chat para tirar dúvidas. Esse projeto foi desenvolvido a partir da necessidade dessas informações por vários setores da empresa na qual atuo,porém não sendo utilizada oficialmente. Utilizado python puro e a biblioteca tkinter para a interface grafica e a biblioteca sqlite para a integração ao banco de dados

  • Controle de Estoque

    CRUD desenvolvido em Python para gerencia de estoque com entradas e saídas integrado com banco de dados sqlite3. Cadastro de novos produtos, edição de entradas e saídas, consulta do saldo de estoques. Exportação da ficha de estoque no formato de excel.

  • Simulação de financiamento SAC/PRICE

    projeto desenvolvido para simular o financiamento nos sistemas SAC e/ou PRICE a partir do valor financiado, taxa de juros mensal e quantidade de meses trazendo o montante, o valor do juros, o valor da parcela, a amortização mensal e o totalizador do valor pago ao final e do valor de juros pago ao final.

    Planilha e Script em VBA criados para consolidar várias planilhas com o mesmo layout em uma só
